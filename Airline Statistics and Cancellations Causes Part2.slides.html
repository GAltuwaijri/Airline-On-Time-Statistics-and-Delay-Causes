Dataset Overview¶

The data consists of flight arrival and departure details for all commercial flights within the USA, The data is separated by year. We decided to explore only the flights on 2008 because it was the nearest year that contained a full data files.

I'll probably be using many of the variables in the data set like Month, DayofMonth, DayOfWeek, UniqueCarrier, Distance, Cancelled, CancellationCode. Others like delay related features, fligth number and taxi times don't seem like they'll be cery useful for me.
